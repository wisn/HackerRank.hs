# Solve Me First in Ruby
Go to [Solve Me First](..) problem.

```ruby
a = gets.chomp.to_i
b = gets.chomp.to_i

result = a + b

puts result
```

## Explanation
1. Request an input (Int) from user and assign as `a`
2. Request an input (Int) from user and assign as `b`
3. Sum `a + b` and assign it to `result`
4. Print the `result`
